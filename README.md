# 🎓 Student Management System

A web-based CRUD application built using Flask and SQL Server.

## 🚀 Features
- Admin Login
- Add Student
- Search Student (Partial match)
- Delete Student
- Dashboard View
- SQL Server Integration

## 🛠 Tech Stack
- Python
- Flask
- SQL Server
- HTML / CSS
- pyodbc

## 🗄 Database
Database managed using SQL Server Management Studio (SSMS)

## ▶ How to Run
1. Clone the repository
2. Install dependencies:
   pip install flask pyodbc
3. Configure SQL Server connection
4. Run:
   python app.py
